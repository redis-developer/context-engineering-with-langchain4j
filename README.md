# Context Engineering with LangChain4J Workshop

## Overview

In this workshop, developers will learn how to implement context engineering with the LangChain4J framework. They will dive deep into the support that LangChain4J provides to create a dynamic context that can significantly improve how LLMs provide correct answers.

## Tech Stack

- **Backend**: 
  - Java 21
  - Spring Boot
  - OpenAI API
- **Frontend**:
  - React
  - HTML5 & CSS3
  - Axios
  - Nginx (for serving the React app)
- **DevOps**:
  - Docker
  - Docker Compose

## Prerequisites

Before you begin, ensure you have the following installed on your machine:

- [Docker](https://www.docker.com/products/docker-desktop)
- [Docker Compose](https://docs.docker.com/compose/install/)
- [Java 21](https://adoptopenjdk.net/) (if running the Spring Boot app locally)
- [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) (if running the React app locally)
